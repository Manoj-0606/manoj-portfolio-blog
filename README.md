# Personal Portfolio Website with Blog Management System

## Overview

This project is a personal portfolio website that showcases my skills, projects, and experience. It also includes a blog management system with an admin portal for managing blog posts.

## Features

### Public Portfolio

* Home page with name, bio, and skills
* Projects page displaying multiple projects
* Blog page listing all blog posts
* Individual blog page to view full content

### Admin Portal

* Login with hardcoded credentials
* View all blog posts
* Add new blog posts
* Edit existing blog posts
* Delete blog posts
* Mark posts as Published or Draft

## Tech Stack

* HTML
* CSS / Tailwind CSS
* JavaScript
* Vercel (Deployment)
